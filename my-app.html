# Users.Login
Description : Log a user into the system. After successful login, system return a uniquely generated token to be used to access all API's functions.
URL : /users/login
Method : POST
Headers : {"Content-Type"=>"application/json"}
Parameters : login: User ID, string, mandatory
password: Password, string, mandatory
Examples :

Login using wrong user ID

URL:
http://localhost:4567/users/login

Parameters:
{"login":"invalid_user","password":"invalid_password"}

Output:
{
  "code": -1,
  "desc": "Invalid login and/or password.",
  "data": null
}

Login using wrong password

URL:
http://localhost:4567/users/login

Parameters:
{"login":"superadmin","password":"superduper"}

Output:
{
  "code": -1,
  "desc": "Invalid login and/or password.",
  "data": null
}

Login as Admin

URL:
http://localhost:4567/users/login

Parameters:
{"login":"admin","password":"admin"}

Output:
{
  "code": 0,
  "desc": "success",
  "data": {
    "token": "08e77ae7c7ea199c6ada25e690fa4490",
  }
}

# Users.List
